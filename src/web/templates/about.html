JRA競馬予想アプリについて

概要

このアプリケーションは、JRAの競馬データを収集し、機械学習モデルを使用して レース結果を予想するWebアプリケーションです。

技術スタック

  • バックエンド: Python, Flask
  • データベース: PostgreSQL / SQLite
  • 機械学習: scikit-learn, XGBoost
  • データ処理: pandas, numpy
  • Webスクレイピング: BeautifulSoup4
  • フロントエンド: Bootstrap 5

主な機能

  • JRAウェブサイトからのレースデータ自動収集
  • 過去のデータに基づく機械学習モデルの訓練
  • レース結果の予想と信頼度スコア表示
  • 馬・騎手・調教師の詳細情報
  • 予想精度の統計分析
免責事項

このアプリケーションの予想は参考情報です。予想の正確性を保証するものではありません。 実際の馬券購入は自己責任で行ってください。また、ギャンブル依存症にご注意ください。

データソース

すべてのレースデータはJRA（日本中央競馬会）のウェブサイトから取得しています。 データの正確性については、JRA公式サイトをご確認ください。
